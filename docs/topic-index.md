# Topic Index — Integrated Musicianship: Theory

A complete index of all topics across all 22 chapters, organized for reference and potential rearrangement. Each entry includes the file path, topic title, file type, and a brief description.

**File types:**
- **Lesson** (`a1-`, `b1-`, etc.) — authoritative instructional text written by Sean Butterfield
- **Discussion** (`a2-`, `b2-`, etc.) — student-curated lecture notes + Open Music Theory excerpts
- **Reading** (`1-`) — supplementary readings, mostly adapted from Open Music Theory

## Content Status Notes

**Chapters 1-12:** Lesson files are fully written with substantial prose, ABC notation examples, and conclusion sections. These represent the earliest writing and are candidates for style revision.

**Chapters 13-22:** Many Lesson files are empty or contain only ABC notation examples without prose. The Discussion files carry most of the instructional content for these chapters. Chapter 14a and 16a are notable exceptions with fully developed Lesson prose. Chapter 19a has detailed Lesson prose with real-repertoire analyses (Schubert, Schumann, Jobim).

**Chapter 13:** Largely empty — most files are placeholders. The Discussion files reference Open Music Theory content on periods and sentences.

**Naming convention shift:** Chapters 1-12 use `a1-`/`a2-` naming. Chapters 13+ use `-ex-`/`-tx-` suffixes to distinguish Lesson from Discussion files.
